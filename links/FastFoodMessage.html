FastFoodMessage, 2022

interactive playful experience

How many seconds do you need to send a message? How many seconds do you need to heat up the food? FastFood Message is a microwave interface with a playful experience. The experience that the user is engaged in is to send a message in 10 seconds. This work is an irony of the rapid communication where messages are usually intermittent nowadays. This puts a user in this stressful position with a countdown timer in order to send it to a receiver. Rethinking a microwave as an interface is making a user in the meditative position of waiting in order to get something very fast and easy.
